# 概率统计学习资料

系统化的概率论与数理统计学习指南，专为"听书"场景设计。

## ✨ 特点

- 📖 **可听性设计**：每个公式都有朗读版本
- 🎯 **直观理解优先**：用生活例子解释抽象概念  
- 🔢 **完整数学内容**：高等数学级别，公式不省略
- 🤖 **AI应用专章**：交叉熵、VAE、扩散模型等

## 📚 内容大纲

### 第一部分：随机事件与概率
- 随机现象与样本空间
- 概率的定义与性质
- 条件概率与独立性

### 第二部分：随机变量及其分布
- 随机变量的概念
- 常见离散分布（二项、泊松、几何）
- 常见连续分布（正态、指数、均匀）

### 第三部分：多维随机变量
- 联合分布、边缘分布、条件分布

### 第四部分：数字特征
- 期望、方差、协方差、相关系数

### 第五部分：大数定律与中心极限定理

### 第六部分：数理统计基础
- 参数估计（矩估计、最大似然）
- 假设检验（Z检验、t检验）

### 第七部分：AI与概率统计
- 神经网络训练的概率本质
- 变分推断与VAE
- 扩散模型的数学基础
- 贝叶斯优化
